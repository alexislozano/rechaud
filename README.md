# rechaud

Renamer for shows and anime

Before:
```
Shows
    The Office
        Season 1
            01 - First episode
            02 - Second episode
        Season 2
            01 - First episode
            02 - Second episode
    Big Bang Theory
        Season 1
            FirstEpisode
            FourthEpisode
            SecondEpisode
            ThirdEpisode
```

After:
```
Shows
    The Office
        S01
            S01E01
            S01E02
        S02
            S02E01
            S02E02
    Big Bang Theory
        S01
            S01E01
            S01E02
            S01E03
            S01E04
```

In order to work even if the files are named incorrectly and thus mixed (for instance first episode not being the first file in the folder), the tool will let you change the episodes / seasons order if needed.
